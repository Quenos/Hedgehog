# Getting started

After installing and building as shown below you can start the program by going to dist folder and start 'Hedgehog-vue.exe' for windows and Hedgehog-vue for mac.
The first time you run the program you have to enter your API keys in the API menu which slides in from the left when clicking the hamburger on the left of the title bar.

Click the orange '+' sign and enter your API keys.
Although it is possible to add more than one API key set, only the first one you entered will be used.

# Known issues:

After entering the API keys it is sometimes necessary to restart the program
